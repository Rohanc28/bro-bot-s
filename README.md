## Commands:
Bro BOT looks for prefix "bro" to detect messages. Some commands it responds to.
```
on?
bot ip
bro help
bro roll
bro comic
bro joke
bro trivia
bro coin
covid
github
sup/Sup
```
## Type "bro help" to know all current commands the bot replies to
```
bro help
```
